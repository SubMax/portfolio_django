Описание проекта

Проект предназначен для получения и хранения основной информации об инвестиционных инструментах, торгуемых на бирже в New York.

Информация об акциях получаем с помощью проекта yfinance из Yahoo! Finance.

Внешний вид проекта построен на основе Bootstrap с использованием BootstrapCDN. Построение графиков цен с помощью Chart.js с использованием Chart.js CDN.
